# Passes — журнал проходов по репозиториям

Датированные отчёты о каждом «проходе» (исследование / правка / аудит) по
`FedorMilovanov/gb-is-my-strength` и связанным. Цель — **память проекта**: чтобы
не терять контекст между сессиями, знать, что уже исследовано, что сделано, где
что лежит, и не начинать каждый раз с нуля.

## Соглашение

- Один файл на проход: `passes/ГГГГ-ММ-ДД-тема.md`.
- Внутри — по шаблону: **Дата · Тема · Ветка/коммиты · Что нашли · Что поменяли ·
  Карта файлов · Открытые вопросы / что дальше**.
- Ссылки на дизайн-эталоны — в `references/` (для Gill: `references/gill-mobile/`,
  канон — `gill-mobile-bars-v2.7.html`).
- Когда проход продолжается в новой сессии — дописываем тот же файл или заводим
  новый со ссылкой на предыдущий.
