# Archive

This folder stores **historical / superseded / template / baseline** material that should not sit in active roots.

## 2026-06-27 cleanup buckets
- `2026-06-27-working/` — superseded working syntheses, old bug matrices, placeholder/history-only notes
- `2026-06-27-verified/` — older verified ledgers, repair orders, templates, old START_HERE
- `2026-06-27-verification/` — superseded verification entry/docs

## Rule
