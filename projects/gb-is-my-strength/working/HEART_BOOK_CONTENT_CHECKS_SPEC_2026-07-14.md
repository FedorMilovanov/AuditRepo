# Спецификация контент-чеков — онлайн-книга «Тайны человеческого сердца»

> **Статус:** working-спека (2026-07-14), кодирование чекера — отдельная задача.
> **SSOT плана книги:** Research-репо, `СЕРИЯ СЕРДЦЕ/61_BOOK_ARCHITECTURE_V2_CHAPTERS_AND_RESEARCH_TASKS.md`
> (10 глав, приоритеты P0–P3, research-задания R1–R8+). Этот файл НЕ дублирует план —
> только проверяемые инварианты для будущих статей серии hard-texts.

## Зачем
Серия превращается в книгу (~38 статей). Ручная сверка не масштабируется. Ниже —
инварианты, которые можно проверять грепом/скриптом на MDX-исходниках
`src/content/articles/*.mdx` (owner-фильтр: `series: "hard-texts"`).

## Чеки

### HB-QUOTE-01 — атрибуция вторичных цитат
Каждый блок цитаты (`> …` с именем автора в хвосте/подписи) обязан содержать источник:
труд + раздел/страница. Грепаемый маркер отсутствия: имя из списка авторов серии
(Owen|Оуэн|Edwards|Эдвардс|Chalmers|Чалмерс|Mead|Мид|Boston|Бостон|Watson|Уотсон|
Burroughs|Берроуз|Flavel|Флавел|Сперджен|Spurgeon|Гилл|Gill) рядом с цитатой без
скобочной ссылки на труд. Severity: P1.

### HB-QUOTE-02 — стоп-лист неверифицированных атрибуций
Запрещено публиковать как прямую цитату без верифицированного первоисточника:
- «splendida vitia» / «блестящие пороки» как дословная цитата Августина (атрибуция спорная);
- M'Cheyne «на каждый взгляд на себя — десять взглядов на Христа» (источник не установлен);
Допустимая форма — пересказ с пометкой природы источника («формула, закрепившаяся в
традиции»). Чекер: греп по паттернам + требование слова «традиц|приписыва» в ±2 строки.
Severity: P0 (ложная атрибуция = удар по доверию всей серии).

### HB-ANCHOR-01 — якорь «Слово + вера + Дух» (принцип R7)
Статьи о перемене/борьбе (список слагов ведётся в конфиге чекера; старт:
kak-menyaetsya-serdce, kak-hranit-serdce, будущие dve-borby, rozhdenie-svyshe) обязаны
содержать связку: упоминание Писания/Слова + веры + Духа как средства перемены
(греп: `Слов[оае]` И `вер[ауыое]` И `Дух` в одном разделе) и НЕ подавать перемену как
технику/психологию. Severity: P2 (редакторский).

### HB-ANCHOR-02 — якорь «созерцание первично» (принцип R8)
Статьи о дисциплинах (kak-hranit-serdce, serdce-i-iskushenie, будущие главы VII) обязаны
含 отсылку к 2 Кор. 3:18 / созерцанию Христа как источнику силы (греп: `3:18|взира|созерца`).
Severity: P2.

### HB-BALANCE-01 — многогранный Христос (принцип R9)
Статья «Христос — Пленитель сердца» (и любая статья, где Христос — главный предмет)
обязана содержать обе грани: милость (Мф. 11:29 / «Сердце Христа») И величие/суд
(Откр. 1 / Откр. 19 / Евр. 4:13). Греп: `Откр` в статьях с `Пленитель|кроткий и смиренный`.
Severity: P1 (запрос owner: не «добрый дядя»).

### HB-TERM-01 — регистр «Дух»
Вне цитат Синодального перевода: «Святой Дух»/«Дух» (Личность) — с заглавной; человеческий
дух — со строчной. Известные легитимные строчные: цитаты Рим. 8:1–15 по Синодальному,
Рим. 8:10 (дух человека), «дух рабства» (метафора состояния). Чекер: греп `дух` с
заглавным контекст-словом (Святой|Божий|усыновления|жизни) → флаг, если строчная.
Severity: P1.

### HB-SRC-01 — блок «Источники и сверка»
Каждая новая статья серии обязана иметь H2 «Источники и сверка» с ≥5 позициями,
из них ≥1 первоисточник эпохи (пуритане/реформаторы) с рабочей ссылкой
(ccel.org|monergism|digitalpuritan|archive.org|gutenberg). Severity: P1.

### HB-STRUCT-01 — соответствие плану книги
Слаг статьи обязан существовать в актуальной версии плана (файл 61 в Research) с той же
главой-привязкой. Дрейф план↔сайт — самый вероятный источник хаоса при 38 статьях.
Severity: P1. (Реализация: экспорт таблицы плана в JSON рядом с чекером.)
