## «Оглавление части» — аккордеон (файл mobile-toc-accordion-v5.html)
Утверждённый интерактивный эталон. Ключевые решения:
- Текущая часть раскрыта по умолчанию; каждая часть сворачивается/раскрывается независимо;
  много частей (10–20) — просто скролл с фейд-масками.
- Выделение текущей: МИНИМАЛИЗМ — мягкая тёплая заливка + тонкая золотая полоса слева
  с лёгким свечением. Никаких рамок/бордеров-карточек.
- Форзацы (Введение/Пролог/Справочник): слово ВЕРТИКАЛЬНО, мелким капсом с
  трекингом, во весь рост строки, БЕЗ кружков/пилюль.
- Субменю: рельса + точки (H2 крупнее, H3 мельче), активный раздел — непрозрачная
  пилюля + точка с золотым гало; пройденные — золотые точки; каскадная анимация
  появления строк (стаггер ~30мс), пружинный шеврон.
- В чужой части первая строка — «→ Перейти к части» (иначе аккордеон запирает навигацию).
- Футер: [▶ Слушать часть — золотой primary со стеклянным бликом]
  [Продолжить → — вторичная] [⎙ печать/PDF — только иконка]. Одинокой широкой кнопки НЕ БЫВАЕТ.
- Данные: целиком из config.pages[*].partToc — ничего не дописывается руками.
